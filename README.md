# Algorithm & Data Structure Practice

This repository contains clean and educational implementations of
fundamental data structures and algorithms using Python.

## Contents
- Data Structures
- Dynamic Programming
- Graph-related structures (Union-Find, Trie, Segment Tree)

## Purpose
- Strengthen algorithmic thinking
- Prepare for technical interviews
- Provide reusable reference implementations

## Target Audience
- Computer Science students
- Software engineering interview candidates
- Anyone revising core algorithms
