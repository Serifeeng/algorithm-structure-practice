## Deque

A Deque (Double-Ended Queue) is a linear data structure that allows
insertion and deletion of elements from both ends.

### Common Use Cases
- Implementing stacks and queues
- Sliding window problems
- Undo / redo operations
- Palindrome checking

### Time Complexity

| Operation        | Complexity |
|------------------|------------|
| Access           | O(n)       |
| Search           | O(n)       |
| Insert (front)   | O(1)       |
| Insert (rear)    | O(1)       |
| Delete (front)   | O(1)       |
| Delete (rear)    | O(1)       |

### Space Complexity
O(n)
